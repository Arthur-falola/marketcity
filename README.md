# MarketCity

Site d'achat en ligne — archi PHP + MySQL

## Démarrage rapide

1. Configurer la base de données MySQL avec le script migrations.sql
2. Configurer les accès DB dans config/db.php
3. Lancer le serveur Apache/Nginx sur public/
4. Utiliser Tailwind CSS pour le front
